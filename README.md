# Telegram Keyword Monitor Bot

Бот мониторит выбранные чаты по ключевым словам и отправляет уведомления в отдельный чат/канал.

## Главное изменение
Теперь **не нужно настраивать ID чатов через `.env`**:
- чат для уведомлений задается через команду `/bind_target` + пересылка сообщения;
- чаты для мониторинга задаются через `/bind_watch` + пересылка сообщения.

`.env` нужен только для `BOT_TOKEN`.

---

## 1) Быстрый запуск для обычного пользователя (Windows, один `.bat`)

1. Установите Python 3.11+ (с галочкой **Add Python to PATH**).
2. Скачайте проект и распакуйте.
3. Дважды кликните `start.bat`.
4. При первом запуске батник попросит `BOT_TOKEN` и сам создаст `.env`.
5. Бот запустится автоматически.

Что делает `start.bat`:
- создает `.venv`;
- ставит зависимости;
- запрашивает токен (только в первый запуск);
- запускает бота.

---

## 2) Подготовка в Telegram

1. Создайте бота через **@BotFather** (`/newbot`) и получите `BOT_TOKEN`.
2. Добавьте бота в чаты/каналы, которые хотите мониторить.
3. Добавьте бота в канал/чат назначения (куда будут приходить алерты).
4. Для канала назначения дайте боту право публиковать сообщения.

---

## 3) Настройка через команды бота (без env для chat_id)

Откройте личный чат с ботом и выполните `/start`.

### 3.1 Назначить чат для уведомлений
1. Отправьте `/bind_target`.
2. Перешлите боту любое сообщение из нужного канала/чата.
3. Бот сохранит этот chat_id как `target`.

### 3.2 Добавить чат в мониторинг
1. Отправьте `/bind_watch`.
2. Перешлите сообщение из чата/канала, который нужно отслеживать.

### 3.3 Удалить чат из мониторинга
1. Отправьте `/unbind_watch`.
2. Перешлите сообщение из чата/канала, который надо убрать.

### 3.4 Задать ключевые слова

```text
/keywords скидка,распродажа,alert
```

### 3.5 Посмотреть текущие настройки

```text
/list
```

### 3.6 Узнать ID текущего чата

```text
/chatid
```

---

## 4) Linux / серверный запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# впишите BOT_TOKEN
python3 bot.py
```

Дальше настраивайте чаты командами в Telegram (`/bind_target`, `/bind_watch`), как в разделе выше.

---

## 5) Где хранятся настройки

- `BOT_TOKEN` — в `.env`
- все настройки чатов и ключевых слов — в `bot_config.json` (создается автоматически)

---

## 6) Частые проблемы

1. Бот не может отправлять в target-канал:
   - проверьте, что бот админ канала;
   - повторите `/bind_target` и снова перешлите сообщение из нужного канала.

2. Бот не видит сообщения в мониторимом чате:
   - убедитесь, что бот добавлен в чат;
   - повторите `/bind_watch`.

3. Ключевое слово не ловится:
   - проверьте, что слово есть в `/list`;
   - задайте заново `/keywords ...`.
